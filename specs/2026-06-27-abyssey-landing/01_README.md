# Abyssey.ai 랜딩 — 심연 스크롤 첫 페이지

> 가상의 실리콘밸리 AI 스타트업 "Abyssey.ai" 랜딩 페이지. 첫 작업 = 심연 스크롤 Hero.

## 목적 / 배경

- BE+FE 풀스택 루프를 처음 증명하는 첫 실제 기능.
- 동시에 회사 컨셉(Abyssey.ai = Deep Alive Odyssey)의 랜딩 첫 페이지를 만든다.
- 시그니처 UX: **첫 화면에서 스크롤하면 심연으로 하강 → 바닥에 다다르면 삼지창/보석함이 드러남.**
- 톤: **미국(실리콘밸리) 사이트 느낌** — 큰 굵은 타이포, 넉넉한 여백, 다크+시안, 절제된 애니메이션, 명확한 CTA. (한국식 X)

## 범위

- 포함:
  - FE 스캐폴드 (Vite + React + TS + Tailwind) — `frontend/`
  - 심연 스크롤 Hero 1개 (목업을 React로 이식)
  - 나머지 섹션/메뉴는 **구도만** (Platform/Research/Pricing/Company 자리만)
  - 사용자 로고 적용 (PNG, 배경 투명화) + 바닥 일러스트(abyss.png) 적용
- 제외 (지금 안 함):
  - 실제 다중 페이지/라우팅, 인증, 결제, 백엔드 연동 기능
  - 실서비스 배포 (Vercel 등) — 나중
